# 🤖 DocuHelper Forex — AI-помощник для трейдеров

**DocuHelper Forex** — это Telegram‑бот, который отвечает на вопросы о Форекс, трейдинге, аналитике и управлении рисками. Бот использует **семантический поиск** (RAG): он находит релевантные фрагменты в вашей базе знаний и формирует точные ответы с помощью языковой модели. Всё работает 24/7 на бесплатных сервисах.

![Демо](https://i.ibb.co.com/5gkJ48gv/Copilot-20260503-095022.webp)

## 🚀 Возможности

- 📚 **База знаний** — любые текстовые файлы (`.txt`) в папке `knowledge/`. Добавляйте новые материалы — бот переиндексирует их.
- 🧠 **Семантический поиск** — ищет не по ключевым словам, а по смыслу. Использует эмбеддинги Cohere и векторную базу PostgreSQL (pgvector).
- 🤖 **Генерация ответов** — через OpenRouter (бесплатные модели, например `google/gemini-2.0-flash-exp:free`). Ответы строго по вашему контексту.
- 💬 **Удобный интерфейс** — кнопки с частыми вопросами, inline‑клавиатура обратной связи (👍/👎).
- 📊 **Статистика и логи** — все вопросы и ответы сохраняются, есть команда `/stats` для администратора.
- ⚡ **Кэширование эмбеддингов** — ускоряет повторные запросы.
- 🔒 **Безопасность** — команда `/stats` доступна только указанному `ADMIN_ID`.

## 🛠 Технологии

| Компонент       | Технологии                                                                 |
|----------------|----------------------------------------------------------------------------|
| Бэкенд         | Python 3.11, aiogram 3, aiohttp, asyncpg                                   |
